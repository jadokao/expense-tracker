# 家庭記帳本(expense-tracker)

## 簡介
使用者可以：
1. 在首頁一次瀏覽所有支出的清單
2. 在首頁看到所有支出清單的總金額
3. 新增一筆支出
4. 編輯支出的所有屬性 (一次只能編輯一筆)
5. 刪除任何一筆支出 (一次只能刪除一筆)
6. 在首頁可以根據支出「**類別**」篩選支出；總金額的計算只會包括被篩選出來的支出總和。
